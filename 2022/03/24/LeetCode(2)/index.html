重点来了，这道题目的主要解法就是dp
定义dp[i][j]，意思是指下标i到j的字符串是否为回文字符串。
该题目的dp[i][j]分为两种情况：

  • dp[i][j] == true，前提是s[i] == s[j],也就是第i个字符与第j个字符相同。

    • i到j的子串长度小于3，则必为true

    • dp[i+1][j-1] == true,又有前提s[i] == s[j]，所以dp[i][j] == true

  • dp[i][j] == false,其他情况都为false，例如不满足s[i] == s[j]即为false

分析完毕，剩余的工作就是遍历过程中需要记录最长子串的长度，以及i。
