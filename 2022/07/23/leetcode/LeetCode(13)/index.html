重建序列

题目链接

题意

给定一个长度为 的整数数组 ，其中 是范围为的整数的排列。还提供了一个 2D 整数数组 ，其中 是 的子序列。
检查 是否是唯一的最短 超序列 。最短 超序列 是 长度最短 的序列，并且所有序列都是它的子序列。对于给定的数组，可能存在多个有效的 超序列 。

例如，对于 ，有两个最短的 超序列 ， 和 。
而对于，唯一可能的最短 超序列 是 。 是可能的超序列，但不是最短的。
如果是序列的唯一最短 超序列 ，则返回 true ，否则返回false。
子序列 是一个可以通过从另一个序列中删除一些元素或不删除任何元素，而不改变其余元素的顺序的序列。

题解

这题开始做的时候一点思路都没有，之前了解过一点拓扑排序，拓扑排序中重要的是存放节点的入度，队列维护，在这个题目中同样是这样的。拓扑排序建立链表，队列存放下一数字的所有可能

  • 下一数字的数量大于一，表示当前不符合唯一性要求，返回false
  • 下一数字的数量等于一，继续遍历序列，知道队列为空。

例如
根据拓扑排序建立链表

①1->2，当前1的入度为0，2的入度为1
②1->3，当前1的入度为0，3的入度为1
③2->3，当前2的入度为1，3的入度为2
将度为0的节点放入队列，循环遍历队列中元素的链表，并及时将入度为0的节点加入到队列。
